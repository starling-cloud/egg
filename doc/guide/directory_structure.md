# Directory Structure

The directory structure for your Grasshopper project could look like this:

```
project/
└── definitions/
    ├── file-01.3dm                 # The Rhino Model
    ├── file-01.gh                  # The Grasshopper Script
    ├── file-01-params.csv
    └── data/
        ├── data_01.csv
        ├── data_02.json
        └── ...
```
